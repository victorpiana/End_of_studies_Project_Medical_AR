# End_of_studies_Project_Medical_AR
Context: Final-year Computer Science Engineering Master's project (UTBM), developed in collaboration with the Belfort Hospital and medical professionals. This project was awarded the highest mark.  Goal: To create an accessible, immersive application for medical personnel training, focusing on complex surgical or diagnostic procedures.
